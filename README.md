# Blood-Donation-Application

Blood Donation App - Upgraded Version

This is an upgraded Flutter project with many features (no Firebase).
Features included:
- Bottom navigation: Home, Donors, Map, Drives, Profile
- Hero header with image and stats
- Donor registration and list
- Mock map using flutter_map & OpenStreetMap with donor markers
- Donation drives carousel
- Emergency (quick) request form
- Profile with QR code (QrImageView) and image picker
- Provider state management and sample data

How to run:
1. Open this folder in VS Code or Android Studio (folder contains pubspec.yaml).
2. Run `flutter pub get`.
3. (Optional) Run `flutter create .` to create platform folders.
4. Run on an emulator or device: `flutter run`.
